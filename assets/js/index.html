The Structure of a Web Page

HTML elements tell the browser how to display the content. Here is a standard boilerplate:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>
